# 学习 Next.js 的心得

Next.js 是一个基于 React 的全栈框架，它提供了很多开箱即用的功能。

## 主要特点

- 服务端渲染 (SSR)
- 静态站点生成 (SSG)
- API 路由
- 文件系统路由

## 使用体验

使用 Next.js 开发博客系统非常方便，它提供了很多我们需要的功能。特别是：

1. 内置的 Markdown 支持
2. 自动的代码分割
3. 优化的图片组件
4. 零配置的 TypeScript 支持

## 性能优化

Next.js 提供了多种性能优化方案：

- 自动的图片优化
- 字体优化
- 路由预加载
- 静态生成
